# DishPatch
An integrated restaurant service-robot control system with a full-stack web platform.

<img alt="DishPatch Architecture" src="./img/DishPatch.png" />

# Background
Service robots are becoming increasingly common in restaurants and hotels. Commercial products such as **Yunji** and **Pudu** are widely used and the overall solutions are robust and mature. However, for individual developers, these systems are often **expensive**, **closed-source**, and **difficult to customise**.

DishPatch aims to solve this by providing an **open-source**, **cloud-based**, **end-to-end service robot system** spanning **frontend, backend, and robotics**. The goal is to make it possible for individual developers (with some software/robotics background) to build and experiment with a complete system—whether at home or in real projects.
